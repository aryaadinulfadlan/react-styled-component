Hallo apa kabar semuanya? semoga sehat selalu ya! Pada kesempatan kali ini saya ingin sharing tentang salah satu library CSS in JS, yaitu styled-components. CSS in JS atau disebut juga JSS  merupakan suatu cara agar kita bisa menulis kode CSS didalam file JavaScript. Tujuannya agar penulisan kode CSS kita sedikit lebih "sakti" dari biasanya. Hal ini tentunya dengan tetap mempertahankan fungsionalitas CSS itu sendiri. Styled-Components merupakan salah satu library JSS yang cukup populer, setidaknya ada lebih dari 50 library JSS.
Ada beberapa keuntungan saat kita menggunakan styled-components, diantaranya kita tidak perlu pusing2 lagi mikirin nama class untuk sebuah element. Nama class akan di generate secara otomatis dan bersifat unik, artinya tidak akan nama class yang sama antar element. Kemudian karena code CSS ini ditulis didalam file JavaScript, artinya kita juga bisa melakukan pengkondisian dalam styling element. Oke langsung aja kita coba ya!

Instalasi
Dalam pembahasan kali ini saya akan menggunakan styled-components pada ReactJS, pertama kita install dulu reactjs nya menggunakan CRA dengan command berikut:
npx create-react-app react-styled-component
Setelah instalasi react berhasil masuk ke direktori project react, dan install styled-component dengan command berikut:
cd react-styled-component
npm install --save styled-components

GlobalStyles
Hal pertama yang perlu kita ketahui dalam menggunakan styled-component adalah GlobalStyles. GlobalStyles adalah cara agar kita bisa menerapkan style secara global didalam aplikasi yang dibangun, artinya style ini akan berlaku diseluruh component (jika tidak ada everriding style). Hal ini dapat kita manfaatkan untuk melakukan CSS reset agar kita memiliki kendali penuh terhadap styling aplikasi react.
